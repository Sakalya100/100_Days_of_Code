📌𝐀𝐩𝐩𝐫𝐨𝐚𝐜𝐡

✅ XOR of zero and some bit returns that bit i.e. x^0 = x...

✅ XOR of two same bits returns 0 i.e. x^x = 0...

✅ And, x^y^x = (x^x)^y = 0^y = y...

✅ XOR all bits together to find the unique number.
